# Sky Recharge - Mobile Recharge Website

A modern mobile recharge website built with HTML, CSS, and JavaScript.

## Project Structure

### Day 1: Requirements
- `requirements.txt` - Project requirements and specifications

### Day 2: HTML Structure
- `index.html` - Landing page
- `login.html` - Login page
- `register.html` - Registration page
- `dashboard.html` - Dashboard with plans

### Day 3: HTML + CSS
- All HTML files from Day 2
- `style.css` - Complete styling with animations

### Day 4: Validations & JavaScript
- All files from Day 3
- JavaScript validations for forms
- User authentication logic
- Payment processing

### Day 5: Mock API Integration
- All files from Day 4
- `api.js` - Mock API for backend simulation
- Complete integration with loading states

## Features
- User Registration & Login
- Dashboard with Data Plans
- Payment Processing
- Form Validations
- Mock API Integration
- Responsive Design
- Animations & Transitions

## Technology Stack
- HTML5
- CSS3
- JavaScript (ES6+)
- localStorage for data persistence

## How to Run
Simply open `index.html` in any modern web browser.
